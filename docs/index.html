How it works

Three seconds. Zero files.

1

Press Cmd + Shift + 6

From anywhere. BlixtSnap lives in your menu bar and listens for the global shortcut.

2

Drag to select a region

Native macOS crosshairs appear. Select exactly what you need for context.

3

Paste into Claude Code

The screenshot is already on your clipboard. Hit Cmd + V and keep prompting.

Why BlixtSnap

Less friction, more context

Clipboard only

Screenshots never touch your filesystem. No files to clean up, no Desktop clutter.

Floating preview

A brief thumbnail appears so you can confirm the capture before pasting.

Native speed

Uses macOS screencapture under the hood. No Electron. No overhead.

Menu bar app

Sits quietly in your menu bar. No dock icon. No windows. Just there when you need it.

Get started

Build in 30 seconds

# Clone and build
git clone https://github.com/milesfitzgerald/BlixtSnap.git
cd BlixtSnap
brew install xcodegen
xcodegen generate
open BlixtSnap.xcodeproj
# Press Cmd+R in Xcode

Requires macOS 13+ and Xcode 15+. Grant Screen Recording permission on first capture.
